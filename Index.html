[PRD] 은행수나무 인증삽목묘 직거래 및 재고 관리 플랫폼 1. 개요 (Overview) 제품명: 은빛숲(가칭) - 자동화 온실 기반 은행수나무 인증삽목묘 직거래 플랫폼 배경: 자동화된 비닐하우스(스마트팜)에서 고품질로 균일하게 육묘된 은행수나무 인증삽목묘를 개월생별(3, 4, 7, 12, 18개월)로 분류하여, 조경업자·농가·일반 소비자에게 투명하게 직거래로 판매하고 재고를 관리하고자 함. 목표 (Goals): 개월생별 묘목의 생육 상태(사진/데이터)와 가격을 직관적으로 보여주는 렌더링 페이지 구축 실시간 스마트팜 육묘 환경(온도, 습도 등)을 시각화하여 품질에 대한 신뢰도 확보 깃허브(Github) 저장소를 기반으로 버셀(Vercel)에 배포 가능한 가볍고 빠른 웹 프론트엔드 구현 2. 타겟 사용자 및 핵심 가치 (User Persona & Value Proposition) B2B 고객 (조경업체, 지자체): 대량 구매를 위해 12개월생, 18개월생 등 규격화된 고품질 묘목의 대량 재고 확보 여부가 중요함. B2C 고객 (농가, 귀농인, 취미 농업): 3개월생, 4개월생 등 초기 비용이 저렴한 묘목을 선호하며, 정성적인 생육 상태 정보가 필요함. 관리자 (생산자): 비닐하우스 내부에서 개월생별 재고 수량을 쉽게 업데이트하고 주문을 확인할 수 있어야 함. 3. 핵심 요구사항 (Key Requirements) 3.1. 상품 카테고리 구성 (개월생별 세분화) 묘목의 특성상 연령별로 규격과 용도가 다르므로 다음과 같이 명확히 구분하여 노출합니다. 유묘 단계 (초기 집중 육묘): 3개월생 / 4개월생 (포트묘 중심, 초기 활착용) 정착 단계 (중기 육묘): 7개월생 (노지 이식 적기) 성묘 단계 (완성형 묘목): 12개월생 / 18개월생 (즉시 식재 가능, 인증필 완료) 3.2. 핵심 페이지 및 기능 (Features) 메인 페이지 (Landing Page): 자동화 비닐하우스 전경 및 인증삽목묘 브랜드 소개 실시간 스마트팜 환경 지표 모킹 데이터 노출 (온도 24°C, 습도 60% 등) 상품 목록 및 상세 페이지 (Product List & Detail): 3, 4, 7, 12, 18개월생별 필터 기능 각 개월생별 평균 수고(키), 근원직경, 뿌리 발달 상태(사진) 정보 제공 대량 구매 문의 버튼 (Call to Action) 대시보드형 관리자 페이지 (Admin - MVP 단계에서는 모킹 또는 간단한 수정 기능): 개월생별 재고 수량 조정 기능 (UPDATE 쿼리 기반 구조) 4. 기술 스택 및 배포 요구사항 (Tech Stack & Deployment) AI가 코드를 생성할 때 Vercel 환경에 최적화하도록 지정하는 기술 규격입니다. Frontend Framework: Next.js (App Router) 또는 React Styling: SCSS Modules 및 부드러운 호버 애니메이션(마이크로 인터랙션)을 위한 classNames 라이브러리 활용 Database/Backend (선택 사양): 서버리스 환경에 적합한 Google Firebase (Firestore) 연동 구조 설계 Deployment Workflow: * 소스 코드는 Github 전용 리포지토리에 push Vercel 프로덕션 환경에 자동 CI/CD 배포 (*.vercel.app 도메인 생성 무결성 확인) 5. 단계별 마일스톤 (Milestones) 1단계 (디자인 및 스키마): 개월생별 상품 테이블 구조(ERD) 정의 및 UI 와이어프레임 확보 2단계 (AI 코드 생성): AI Studio/Cursor를 활용한 3, 4, 7, 12, 18개월생 필터링 페이지 및 상품 카드 컴포넌트 생성 3단계 (배포): Github 연동 후 Vercel 최초 배포 및 모바일 웹 뷰포트 최적화 반응형 확인
